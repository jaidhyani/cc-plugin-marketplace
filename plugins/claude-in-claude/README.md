# claude-in-claude

Test Claude Code programmatically with structured output and session management.

This plugin provides tools for automated testing of Claude Code CLI interactions. It handles structured JSON parsing of Claude's output, manages multiturn sessions via `--resume`, and includes isolation patterns for running tests in CI/containers without side effects.

## Features

- Programmatic CLI testing with JSON output parsing
- Multiturn session management and resumption
- Isolation patterns for CI and containerized environments
- Test automation skills
