# Ultrapowers

Opinionated dev workflow: brainstorming, debugging, TDD, verification, planning, code review, and agent coordination.

## Skills

- **agents** - Coordinate and manage multiple agents
- **brainstorm** - Ideate and explore solutions
- **code-review** - Review and critique code changes
- **debugging** - Diagnose and fix issues
- **plans** - Create and track development plans
- **tdd** - Test-driven development workflow
- **verify** - Validate implementations and test coverage
